# DesktopFileCreator
## A python script to make creating desktop files easier!

I've decided the unique linux users would enjoy a script to make Desktop files easier. Feel free to improve on it!

For extra information the install.sh file is optional you don't need to use it. You need to make it executable `chmod +x install.sh`

What it does:

- Installs [PySide6](https://en.wikipedia.org/wiki/PySide)
- Makes the Python file executable
- Moves the script into a folder in your home directory (~/Desktopfilecreator)
- Creates a .desktop launcher for easy access
